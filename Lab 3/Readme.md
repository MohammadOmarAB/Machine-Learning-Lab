# Lab 3 -- Car Sales Analysis

## Files

-   `Lab 3.ipynb` -- Jupyter Notebook with all code and analysis\
-   `Car_sales.csv` -- Dataset used in the notebook

## Description

This lab analyzes a car sales dataset using Python.\
The notebook includes data loading, basic cleaning, summary statistics,
and simple visualizations.

## Dataset Description - Car_sales.csv

This dataset contains sales and specification data for different car models. It includes information such as manufacturer, model name, sales figures, price, engine size, horsepower, fuel efficiency, and other vehicle features.

The data is used to analyze trends in car sales and explore relationships between vehicle characteristics and performance in the market.

## How to Run

1.  Make sure both files are in the same folder.\
2.  Open `Lab 3.ipynb` in Jupyter Notebook or Google Colab.\
3.  Run all cells in order.

## Tools Used

-   Python\
-   pandas\
-   matplotlib / seaborn
