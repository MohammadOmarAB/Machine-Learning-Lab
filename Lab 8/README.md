### K-Nearest Neighbors (KNN) Lab

#### Overview
This lab demonstrates how to implement the K-Nearest Neighbors (KNN) algorithm using Python and Scikit-learn.

#### Steps Covered
1. Load and explore the dataset
2. Perform exploratory data analysis (EDA)
3. Standardize features
4. Split data into training and testing sets
5. Train a KNN model
6. Evaluate model performance
7. Optimize K using the elbow method

#### How to Run
1. Place the dataset file `KNN_Project_Data` in the same directory as the notebook.
2. Open the notebook in Jupyter Notebook or VS Code.
3. Run all cells sequentially.

#### Notes
- Feature scaling is important for KNN since it relies on distance metrics.
- The optimal K value is selected using the error rate plot.
