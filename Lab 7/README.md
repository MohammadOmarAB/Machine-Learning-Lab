### Logistic Regression Assignment

#### Overview
This notebook demonstrates how to build and evaluate a Logistic Regression model using a real-world advertising dataset. The goal is to predict whether a user will click on an ad based on given features.

#### Dataset
The dataset (`advertising.csv`) includes features such as:
- Daily Time Spent on Site
- Age
- Area Income
- Daily Internet Usage
- Ad Topic Line
- City
- Male
- Country
- Timestamp
- Clicked on Ad (target variable)

#### Model
A Logistic Regression model from `sklearn` is used to perform binary classification.

#### Results
The model is evaluated using precision, recall, and F1-score. Results may vary slightly depending on the environment and library versions.
